Підстановка (entity). Це досить потужний інструмент генерації HTML-сторінки на стороні браузера. Підстановки використовуються в якості значень атрибутів HTML-тегів. Наприклад, як значення за замовчуванням поля форми, що визначає домашню сторінку користувача, буде вказано URL поточної сторінки:

<INPUT TYPE = button VALUE = "entity" onClick = "window.alert ( 'Будь ласка, натисніть ще раз');">
