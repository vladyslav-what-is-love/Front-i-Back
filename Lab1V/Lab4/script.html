Сценарій на JavaScript включається в документ за допомогою тега script

<head> <script> <!-- ... оператори сценарію ... --> </script> </head>

Розміщення сценарію в розділ head документа призводить до того, що сценарій буде завантажений до того, як буде потрібно його виконати.

Код сценарію розміщують в теги коментаря html для того, щоб старі браузери, які не розуміють JavaScript, не відображали цей код на екрані.
