Дочерний селектор

Дочірнім називається елемент, який безпосередньо розташовується всередині батьківського елементу.

<style> DIV I { /* Контекстный селектор */ color: green; /* Зеленый цвет текста */ } P>I { /* Дочерний селектор */ color: red; /* Красный цвет текста */ } </style> <div> <p><i>Lorem ipsum dolor sit amet</i>, consectetuer adipiscing elit. <i>dolore magna</i> aliguam erat volutpat.</p> </div>
