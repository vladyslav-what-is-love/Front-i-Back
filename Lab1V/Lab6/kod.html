що перетягується. </li> <li>Resizable - змінює розмір HTML-елемента за допомогою мишки. </li> <li>Selectable - використовується мишка для виділення елемента або групи елементів.</li> <li>Sortable - <li>Autocomplete - автодоповнення введення по перших символах. </li> <li>Button - кнопки на сторінці. </li> <li>Datepicker - вибір дати з календаря. </li> <li>Dialog - створення діалогових вікон. </li> <li>Menu – випадне меню.</li> <li>Progressbar - прогрес виконання деякого процесу. </li> <li>Selectmenu - розширює функціональність базового HTML-елемента. </li> <li>Slider - вибір числа за допомогою повзунка.</li> <li>Spinner - для зміни значень в текстовому полі введення. </li> <li>Tabs - організація вкладок на сторінці. </li> <li>Tooltip - організація спливаючих підказок. </li> </ul> </div> <h3>Ефекти (Effects) jQuery UI </h3> <div> <ul> <li>Add Class - додає CSS-клас елементу. </li> <li>Color Animation - розширення анімації jQuery. </li> <li>Easing - зміна темпу анімації. </li> <li>Effect - анімаційні ефекти. </li> <li>Hide - приховування елемента з використанням анімаційних ефектів. </li> <li>Remove Class - видалення CSS-класу.</li> <li>Show - поява елемента з використанням анімаційних ефектів. </li> <li>Switch Class - перемикання класів. </li> <li>Toggle - по черзі відображає і ховає елемент з використанням анімаційних ефектів. </li> <li>Toggle Class - по черзі додає і видаляє клас з елемента (на відміну від Switch Class працює з одним класом). </li> </ul> </div> <h3>Утиліти (Utilities) jQuery UI </h3> <div> <ul> <li>Position - управління позиціонуванням елементу
